# Focus Fish Tank

A Chrome extension that helps you stay focused with a virtual fish tank. Your fish thrive when you stay on task â€” get distracted and watch them suffer.

## Features

- Virtual fish tank that lives in your browser
- Fish health tied to your focus and browsing habits
- Customizable blocked/allowed site lists
- Session tracking and focus streaks

## Installation

### From Source

1. Clone the repository:
   ```bash
   git clone https://github.com/jackieyang/FocusFishTankChromeExt.git
   cd FocusFishTankChromeExt
   ```
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked** and select the project directory

## Development

### Prerequisites

- Google Chrome (or Chromium-based browser)
- Node.js and npm (if using a build step)

### Getting Started

```bash
npm install
npm run build
```

Then load the `dist/` (or project root) as an unpacked extension in Chrome.
